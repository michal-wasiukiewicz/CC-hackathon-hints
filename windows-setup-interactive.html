WSL2 — Windows Subsystem for Linux Wymagane
Silnik Dockera · Wymagany restart · ~10 min
✓ DONE
▼
Docker Desktop na Windows używa WSL2 jako silnika. Bez WSL2 Docker nie zadziała. Ten krok wymaga restartu komputera — postęp w przeglądarce zostanie zachowany.
Instalacja WSL2
Windows Terminal (Admin)
wsl --install
⚠ Restart wymagany. Po restarcie Windows Terminal otworzy się i dokończy instalację Ubuntu. Zostaniesz poproszony o podanie nazwy użytkownika i hasła dla Ubuntu — wpisz cokolwiek prostego (używane tylko lokalnie).
Po restarcie — konfiguracja
Windows Terminal (Admin) — po restarcie
# Ustaw WSL2 jako domyślną wersję
wsl --set-default-version 2

# Sprawdź wersję Ubuntu (powinno pokazać VERSION 2)
wsl -l -v
Konfiguracja limitów pamięci (16 GB RAM)
# Dla 16 GB RAM — zostaw 6 GB dla Windows i IDE
$wslConfig = "[wsl2]`nmemory=8GB`nprocessors=4`nswap=2GB`nlocalhostForwarding=true"
$wslConfig | Out-File -FilePath "$env:USERPROFILE\.wslconfig" -Encoding UTF8

# Dla 8 GB RAM — zmień na memory=4GB, processors=2

# Zastosuj
wsl --shutdown
Odhacz:
✓
wsl --install wykonany, restart komputera wykonany
✓
Ubuntu uruchomione, nazwa użytkownika i hasło ustawione
✓
wsl -l -v pokazuje Ubuntu z VERSION 2
✓
Plik .wslconfig stworzony z limitami pamięci

foreach ($s in $skills) {

    }
}

# Agenci Java

        Copy-Item -Force "$env:USERPROFILE\everything-claude-code\agents\$a" "$env:USERPROFILE\.claude\agents\"

}

# Commands

Weryfikacja całości Wymagane
Skrypt weryfikacyjny + final checklist · ~5 min
✓ DONE
▼
Uruchom ten skrypt — każda pozycja powinna dać zielone ✓. Zero błędów = gotowy na hackathon.
Skrypt weryfikacyjny — wklej i uruchom w całości
$errors = 0
function Check($name, $cmd, $pattern) {
    try {
        $result = Invoke-Expression $cmd 2>&1 | Out-String
        if ($result -match $pattern) { Write-Host "  ✓ $name" -ForegroundColor Green }
        else { Write-Host "  ✗ $name  → $($result.Trim())" -ForegroundColor Red; $script:errors++ }
    } catch { Write-Host "  ✗ $name  → ERROR: $_" -ForegroundColor Red; $script:errors++ }
}

Write-Host "`n═══ HACKATHON SETUP WERYFIKACJA ═══" -ForegroundColor Cyan
Write-Host "[ Narzędzia ]" -ForegroundColor Yellow
Check "Git"              "git --version"           "git version"
Check "Node.js 22"      "node --version"           "v2[2-9]\.|v[3-9][0-9]\."
Check "npm"             "npm --version"            "[0-9]+\.[0-9]+"
Check "Java 21"         "java -version 2>&1"       "21\."
Check "javac 21"        "javac -version 2>&1"      "21\."
Check "Maven 3.9"       "mvn --version"            "Apache Maven 3\."
Check "Maven + Java 21" "mvn --version"            "Java version: 21"
Write-Host "[ Docker ]" -ForegroundColor Yellow
Check "Docker daemon"   "docker --version"         "Docker version"
Check "Docker Compose"  "docker compose version"   "Docker Compose"
Check "Docker test"     "docker run --rm hello-world 2>&1" "Hello from Docker"
Write-Host "[ Claude Code ]" -ForegroundColor Yellow
Check "Claude Code CLI" "claude --version"         "Claude Code"
$cd = "$env:USERPROFILE\.claude"
if (Test-Path "$cd\settings.json")   { Write-Host "  ✓ settings.json" -ForegroundColor Green } else { Write-Host "  ✗ settings.json BRAK" -ForegroundColor Red; $errors++ }
$nc = (Get-ChildItem "$cd\commands" -EA SilentlyContinue).Count
if ($nc -gt 10) { Write-Host "  ✓ ECC commands ($nc)" -ForegroundColor Green } else { Write-Host "  ✗ ECC commands ($nc)" -ForegroundColor Red; $errors++ }
$na = (Get-ChildItem "$cd\agents"   -EA SilentlyContinue).Count
if ($na -gt 5)  { Write-Host "  ✓ ECC agents ($na)"   -ForegroundColor Green } else { Write-Host "  ✗ ECC agents ($na)"   -ForegroundColor Red; $errors++ }
Write-Host "[ WSL2 ]" -ForegroundColor Yellow
Check "WSL2" "wsl -l -v" "2"
Write-Host ""
if ($errors -eq 0) { Write-Host "  ✅ WSZYSTKO OK — gotowy!" -ForegroundColor Green }
else               { Write-Host "  ❌ $errors błąd(ów) — napraw przed hackathon" -ForegroundColor Red }
Write-Host "════════════════════════════════════`n" -ForegroundColor Cyan
